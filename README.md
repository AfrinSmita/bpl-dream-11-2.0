# CricketVerse

A fun and interactive React project where users can manage their own fantasy cricket team, select players, and manage coins. Users can choose players strategically and track their progress in real time.

---

## Technologies Used

- **React.js** – Frontend UI framework  
- **Tailwind CSS** – Styling and layout  
- **React Icons** – Beautiful icon components  
- **React Toastify** – Notification system  

---

## Key Features

1. **Free Coin System** :
Users can claim free credits (1,000,000 coins) to build their team. The coin balance updates in real-time with notification support.

2. **Available Players List** :
Displays a list of cricket players with their name, role, image, and price for selection.

3. **Player Selection Mechanism** :
Users can add players to their selected team while managing their budget efficiently.

4. **Duplicate Player Prevention** :
Prevents selecting the same player more than once and shows a warning message if attempted.

5. **Player Limit Control** :
A maximum of 6 players can be selected to form a team.

6. **Coin-Based Purchase Validation** :
Players can only be selected if the user has sufficient coins; otherwise, an alert is shown.

7. **View Toggle System** :
- Users can switch between:

    --Available Players view

    --Selected Players view

8. **Remove Selected Players** :
Allows users to remove any selected player from the team at any time.

9. **Add More Players Option** :
Users can return to the Available Players view to select more players.

10. **Real-Time Notifications** :
Interactive toast notifications for all major actions such as:

    - Free credit claimed

    - Player added

    - Duplicate selection

    - Insufficient coins

    - Maximum limit reached
